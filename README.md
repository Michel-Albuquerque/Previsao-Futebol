Projeto que visa utilizar modelos de Machine Learning para realizar a previsão de partidas de futebol.

Há uma apresentação de uma script com um exemplo de como uma base foi retirada, bem como um script com um exemplo da aplicação da metodologia, indo da análise dos dados, passando pelo pré-processamento, até a estimação e otimização dos modelos.

Também há um PDF contendo todo o trabalho realizado no Projeto Final, além de um arquivo PPTX utilizada para a apresentação do trabalho.
